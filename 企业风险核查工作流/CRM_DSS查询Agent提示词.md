# CRM DSS查询Agent提示词

1. 打开目标系统：https://crm.i4px.com/site/home ，若页面加载异常，请自动刷新

2. 点击【4PX组织登录】

3. **关键操作**：界面默认为"扫码登录"，精准点击二维码区域右上角的**"电脑/显示器"小图标**，将模式切换为【账号密码登录】。严禁点击二维码中心、刷新按钮或背景区域

4. 在账号框（工号/邮箱/手机号）输入 111，在密码框输入 111

5. 点击蓝色【登录】按钮

6. 等待页面完全跳转至系统首页

7. 进入【客户运营】→【客户管理】→【我的客户】页面

8. **循环查询（对 ${company list} 中的每个企业名称，依次执行以下步骤 8.1~8.6）**：

   8.1 在【搜索客户】输入框中填入当前企业名称，点击【查询】，等待表格加载完成

   8.2 **表格操作（精准滑动）**：
   - 定位表格底部的横向滚动条
   - 点击并按住滚动条滑块，向右拖动约60%-70%的距离
   - 目标：使【DSS】列显示在屏幕中间区域，确保完全可见
   - 注意：不要拖到最右端，也不要只拖动一点点

   8.3 读取该企业对应的 **DSS** 状态

   8.4 **逻辑判断**：
   - 如果 DSS 状态为【未开通】或【关闭】→ 标记该企业 status 为 "ignored"，detail 为 "已关闭"
   - 如果 DSS 状态为【开通】→ 标记该企业 status 为 "abnormal"，detail 为 "未关闭"
   - 如果查询无结果或出错 → 标记该企业 status 为 "fail"，detail 记录错误原因

   8.5 清空搜索框，等待2秒

   8.6 返回步骤 8.1，处理列表中的下一个企业，直到所有企业查询完毕

9. 不要点击【开通品牌时间】等无关输入框
