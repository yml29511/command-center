# BUA表格提取JSON输出格式（精简版）

## 输出格式

```json
[
  {
    "tableId": "table_1",
    "tableName": "表格名称",
    "headers": ["列名1", "列名2", "列名3"],
    "rows": [
      ["值1", "值2", "值3"],
      ["值1", "值2", "值3"]
    ],
    "totalRows": 2
  }
]
```

## 字段说明

- `tableId`: 表格标识（如 `table_1`）
- `tableName`: 表格名称
- `headers`: 表头数组
- `rows`: 数据行数组（二维数组）
- `totalRows`: 数据行数（不含表头）

## 示例

```json
[
  {
    "tableId": "table_1",
    "tableName": "空间成员列表",
    "headers": ["成员", "云账号", "角色", "加入时间", "操作"],
    "rows": [
      ["4pxtech", "4pxtech", "空间管理员", "2020年5月12日 11:05:44", "所有者"],
      ["董闻(禾名)", "4pxtech:248630", "开发", "2025年12月22日 17:27:12", "移除"]
    ],
    "totalRows": 2
  }
]
```

## 处理规则

1. **必须输出JSON数组** `[...]`，不要输出对象 `{...}`
2. **不要包含Markdown标记**（如 ` ```json `）
3. **多值字段用空格分隔**（如 `"开发 运维 部署"`）
4. **空单元格用空字符串** `""`
5. **只输出JSON，不要有其他文字**
